# Dragon Fighter Prototype

Canvas-only 1v1 dragon duel prototype based on `docs/gdd.md` and `docs/tdd.md`.

## Run locally

```bash
npm test
npm run build
npm run dev
```

Open `http://localhost:5173`.

## Controls

- `A` = Attack
- `D` = Defence
- `B` = Block
- `S` = Skill
- `V` = Voice input when supported by the browser
- `R` = Restart after result

All game visuals, HUD, overlays, buttons, and characters render inside the Canvas. `index.html` is only the Canvas container.
